## فارسی

### اسکنر هوشمند آی‌پی تمیز کلودفلر با خروجی تک‌فایل مستقل ⚡

یک ابزار کلاینت‌ساید (بدون نیاز به سرور) مدرن و بهینه‌سازی شده برای تست تاخیر، سرعت و وضعیت دسترسی به آی‌پی‌های رنج کلودفلر. این برنامه به شما امکان اسکن رنج‌های IPv4 به صورت دسته‌ای را می‌دهد و در نهایت می‌توانید نتایج را علاوه بر فایل متنی `TXT`، به صورت یک نسخه **تک‌فایل HTML مستقل و پرتابل** دریافت کنید که بدون سرور و در هر مرورگر یا دستگاهی (حتی موبایل و به صورت آفلاین) اجرا می‌شود.

#### 🎨 ویژگی‌های کلیدی
- ⚡ **اسکن موازی هوشمند**: بررسی همزمان دسته‌ای (Batch Multi-threading) برای سرعت بالا و بدون فریز شدن مرورگر.
- 📁 **نسخه تماماً آفلاین و پرتابل**: قابلیت دانلود پروژه به عنوان یک تک‌فایل HTML مستقل جهت استفاده روی لوکال یا موبایل.
- ⚙️ **طراحی واکنش‌گرا**: دارای رابط کاربری مدرن با تم تاریک (Dark Mode) و بهینه‌سازی شده برای موبایل و دسکتاپ بر پایه Tailwind.
- 🔒 **۱۰۰٪ امن و کلاینت‌ساید**: تمام تراکنش‌ها و پینگ‌ها مستقیماً از روی مرورگر شما انجام می‌شود.

## English

### Smart Cloudflare Clean IP Scanner with Standalone HTML Export ⚡

A modern, serverless, client-side tool to scan, ping, and check Cloudflare IPv4 ranges. It runs batch-testing directly in your browser and lets you download clean IPs as standard `TXT` lists or as a **fully self-contained, portable, single-file HTML app** that runs offline on any device.

#### 🎨 Key Features
- ⚡ **Smart Parallel Scan**: High-speed batch testing without freezing your UI.
- 📁 **Fully Offline & Portable**: Export the entire tool into one interactive standard HTML file.
- ⚙️ **Modern Design**: Responsive dark-themed container built with Tailwind CSS.
- 🔒 **100% Secure**: All tests happen directly inside your browser cache.
